At the end of October, I saw the above tweet from Topi Tjukanov. Create a map every day in November, each with a different, predetermined theme? Heck yeah! So now I'm making a map every day in November that adheres to the daily theme. I am going to try my best to use a variety of tools and write up an explanation of how each map works, and I'll be posting updates over on Twitter @lyzidiamond with hashtag #30DayMapChallenge.
